# Data-Analytics-FreightFowarding
Full-Stack Data Analysis Project for Analyzing Business KPIs for Freight Forwarders

## Background
ABC Logistics is a forwarder providing international logistics services. The company began implementing a logistics ERP system in October 2024 to transition into a data-driven enterprise, but the vast amount of data within the system had not been fully utilized until now.

## Overview
This repository provides scripts and dashboard for analyzing shipment and accounting data.
The dashboard is designed for management or managers to help data driven decision. The interactive visualizations allow for an in-depth understanding of the business performance and highlight areas for improvement.
